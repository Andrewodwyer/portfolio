Project Overview

Star Squadron is an interactive JavaScript game inspired by the classic Nintendo game StarFox. This project showcases my skills in front-end web development, game design, and interactive user experiences. The game combines character selection, a rock-paper-scissors gameplay mechanic, and space-themed visuals to create an engaging and nostalgic gaming experience.

As the designer and developer, I was responsible for conceptualizing the game, designing the user interface, implementing the game logic, and creating the visual assets. Star Academy demonstrates my ability to create complex, interactive web applications while paying homage to classic video game.

The game features multiple playable characters, each with unique attributes, and a challenging AI opponent. Players must use strategy and quick thinking to outmaneuver their opponent in a series of rock-paper-scissors battles set against a dynamic space backdrop.

User Research & Competitor Analysis

While Star Academy is a personal project inspired by classic games, I still conducted research to ensure the game would be enjoyable and accessible to a wide audience.

User Research Findings

  • Users enjoy games that blend nostalgia with modern gameplay elements
  • Simple, easy-to-understand rules with strategic depth are appealing
  • Character selection and customization enhance player engagement
  • Browser-based games should be quick to load and easy to play without installation

Competitor Analysis:

  • Classic Rock-Paper-Scissors games: Often too simple, lacking depth and replayability
  • Browser-based space shooters: Can be complex and require significant load times
  • Retro game emulators: Legal issues and lack of modern features
  • Mobile casual games: Often rely on microtransactions, which can be off-putting

Based on these insights, I aimed to create a game that combines the simplicity of rock-paper-scissors with the excitement of space combat and character selection, all in an easily accessible browser-based format.

UX/UI Design Process

The design process for Star Academy focused on creating an intuitive and visually appealing game interface that captures the essence of classic space combat games while offering a unique gameplay experience.

  1. Concept Sketching: Created initial sketches of the game layout, character designs, and UI elements.
  2. Wireframing: Developed low-fidelity wireframes to establish the basic layout of the game screens.
  3. Visual Design: Created a cohesive visual style inspired by retro space games, with a modern twist.
  4. Character Design: Designed unique characters with distinct visual styles and attributes.
  5. UI Elements: Crafted intuitive UI components for character selection, game controls, and score display.
  6. Animations: Implemented subtle animations to enhance the game feel and provide visual feedback.
  7. Playtesting: Conducted multiple rounds of playtesting to refine gameplay and identify usability issues.
  8. Iteration: Continuously refined the design based on player feedback and personal insights.

Key Features

  • Character Selection: Choose from multiple unique characters, each with their own strengths and weaknesses
  • Rock-Paper-Scissors Gameplay: Strategic battles based on the classic game mechanic
  • Space Combat Theme: Engaging visuals and animations that bring the space setting to life
  • AI Opponent: Challenging computer-controlled opponent with adaptive difficulty
  • Score Tracking: Keep track of wins, losses, and ties across multiple games
  • Responsive Design: Playable on both desktop and mobile devices
  • Sound Effects: Immersive audio cues for actions and outcomes
  • Tutorial: Built-in instructions to help new players understand the game mechanics

Design Considerations

Colour Palette

The colour scheme was chosen to evoke a futuristic space atmosphere while ensuring good contrast and readability:

  • Primary Colour: Deep Space Blue (#1a2a6c) - represents the vastness of space
  • Secondary Colour: Cosmic Purple (#7303c0) - adds a touch of mystery and excitement
  • Accent Colour: Neon Green (#38ef7d) - used for highlights and important UI elements
  • Background: Dark Gray (#222222) - provides a neutral backdrop for game elements
  • Text: White (#ffffff) - ensures high contrast and readability

Typography

  • Main Font: 'Inter'

Icons & Favicon

  • Game Icons: Custom-designed icons for rock, paper, and scissors actions
  • Character Icons: Portraits for each playable character
  • Favicon: StarFox design that represents the game's theme

Final Thoughts

Developing Star Squadron was an exciting challenge that allowed me to combine front-end development, game design, and interactive storytelling. This project demonstrates my ability to create a fully functional browser-based game with engaging mechanics, responsive design, and a visually appealing interface.

Key takeaways from this project include:

  • The importance of simple yet strategic gameplay to keep players engaged.
  • How visual design and animations enhance the user experience and bring a game to life.
  • The value of browser-based games that are lightweight, quick to load, and require no installation.
  • The power of audio and visual feedback in making interactions feel more dynamic and satisfying.

Moving forward, I plan to expand Star Squadron with additional characters, new battle mechanics, and multiplayer functionality to enhance replayability.
